# clipboard
<Header/>
[[toc]]


### clipboard.readText()
Returns the text saved in clipboard.
* returns `string`

### clipboard.writeText(text)
Writes text to clipboard.
* arguments
  * text `string`
* returns `void`

### clipboard.clear()
Clears the text saved in clipboard.
* returns `void`
