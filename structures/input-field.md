# InputField

[[toc]]

### `placeholder`
Placeholder for the select field.
* type `string`

### `currentValue`
Current value of the select field. Must be one of the values provided in `options` array.
* type `Function`

### `isHidden`
Controls whether the field is visible or hidden.
* type `boolean`

### `handle`
* type `(value?: string) => void`
